## Avant l'exécution de la requête
On a besoin d'un dictionnaire pour ce problème. 

J'ai ordonné chaque mot du dictionnaire en ordre alphabétique pour ensuite les stocker dans une unordered map. 

Les valeurs associés aux clés sont les anagrames possibles.

À noter que le dictionnaire de mots utilisés n'est pas très grand donc la solution marchera pas nécessairement à chaque coup.
